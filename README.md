# Educator–artist building tools, systems, and scenes that let other people build.

## Intent
This is my studio bench: a public set of notebooks, kits, field notes, and working systems for learning in the open. The repos here turn experiments into repeatable practice and usable artifacts without pretending they arrive immaculate. Some are instruments, some are classrooms, some are infrastructure; all of them are meant to be opened, tested, adapted, and carried forward.

This is not a polished product site, a corporate knowledge base, or a shrine to process for its own sake. It is half workshop wall, half teaching guide: practical, documented, a little messy, and built to be picked up. If you want novelty for novelty’s sake, there are louder corners of the internet. If you want utility with some radical honesty, welcome.

**Playful rigor • Open hardware • Repair culture**  
**Humans first • least data • consent by design**

See also: [**PRIVACY & ETHICS**](https://github.com/bseverns/privacy-and-ethics)
